# Agent工程指南

> Agent，默认是LLM-based Agent

随着LLM（large language model）的快速发展，在LLM的加持下，Agent的智能化能力也有了更高的预期。
2024年是Agent火热的一年。Github上出现了很多优秀的Agent开源项目autogen、crewAI、Camel、langgraph等；dify、coze等LLM应用开发平台都新增或者增强了Agent相关能力。

去年，我一直在关注着Agent的进展。今年年初，公司开始了Agent落地场景的探索，我负责了Agent前期调研、研发等全链路过程，整体过程中，我阅读了一些论文以及开源项目。

最近在复盘整体的流程，将自己的学习经历以及工程经验，总结为 Agent工程指南。
